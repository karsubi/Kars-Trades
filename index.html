Notion Template — Copy & Paste Setup

Create a new Notion database called "Trade Journal" with the following properties. Each row = one trade.

Database Properties
📅 Date → Type: Date
⏰ Session → Type: Select → Options: 9:30-11:30 AM / 2:00-3:30 PM / Other
📈 Direction → Type: Select → Options: CALL / PUT
🎯 Strike → Type: Number
📆 Expiration → Type: Select → Options: 0DTE / Next Day / 2DTE
💵 Entry Price → Type: Number
💵 Exit Price → Type: Number
🔢 Contracts → Type: Number
🛑 Stop Loss Level → Type: Number (QQQ price)
🎯 Target Level → Type: Number (QQQ price)
✅ Result → Type: Select → Options: WIN / LOSS / BREAKEVEN
💰 P&L ($) → Type: Formula → (Exit - Entry) × Contracts × 100
🧠 Emotion → Type: Select → Options: Calm / Hesitant / Confident / Anxious / FOMO / Rushed
📋 Checklist Complete → Type: Checkbox
